# TravelXpense

**這是一個專為旅遊打造的記帳軟體，讓你在旅行途中輕鬆管理花費**

### 重點功能
- 旅遊花費紀錄
- 圖表展示各類別花費比例
- 旅遊預算管理
- 帳戶功能
- 當地貨幣轉換
- 資料備份
